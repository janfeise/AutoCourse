## Q&A 常见问题

1. 手机系统不支持软件 auto.js
   可以查看[github 文档](https://github.com/kkevsekk1/AutoX?tab=readme-ov-file)，或者下载安卓模拟器

2. 知到需完成登录

3. 定时任务添加：找到脚本，点击最右侧
